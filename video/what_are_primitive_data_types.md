### What are primitive data types?

Primitive data types in JavaScript are the most basic types of data. They are immutable and include:

- `string`
- `number`
- `boolean`
- `null`
- `undefined`
- `symbol` (ES6)
- `bigint` (ES11)

Example:

```javascript
const name = 'John'; // string
const age = 30; // number
const isActive = true; // boolean
```
