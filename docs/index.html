Levels

Beginner

Fundamental JavaScript concepts: variables, types, operators, basic functions, strings, arrays.

// Example: What is the difference between let and var?
var x = 1;    // function-scoped, hoisted
let y = 2;    // block-scoped, not hoisted
const z = 3;  // block-scoped, cannot reassign

Basic

Core language features: closures, scope, hoisting, this keyword, prototypes, basic DOM manipulation.

// Example: What is a closure?
function outer() {
  let count = 0;
  return function inner() {
    return ++count;
  };
}
const counter = outer();
counter(); // 1
counter(); // 2

Intermediate

Async patterns, promises, event loop, error handling, modules, classes, iterators, generators.

// Example: What is the event loop?
console.log('1');
setTimeout(() => console.log('2'), 0);
Promise.resolve().then(() => console.log('3'));
console.log('4');
// Output: 1, 4, 3, 2

Advanced

Proxies, Reflect, WeakRef, FinalizationRegistry, SharedArrayBuffer, Atomics, advanced patterns, performance optimization, security.

// Example: How does Proxy work?
const handler = {
  get(target, prop) {
    return prop in target ? target[prop] : `Property ${prop} not found`;
  }
};
const obj = new Proxy({ name: 'John' }, handler);
obj.name;  // 'John'
obj.age;   // 'Property age not found'
