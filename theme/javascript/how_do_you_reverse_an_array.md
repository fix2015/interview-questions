### How do you reverse an array?

You can reverse the order of elements in an array using the `reverse()` method in JavaScript. This method modifies the original array in place.

Example:

```javascript
const arr = [1, 2, 3];
arr.reverse();
console.log(arr); // [3, 2, 1]
```
